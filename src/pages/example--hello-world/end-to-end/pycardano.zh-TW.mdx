## 獲取資金

在本教學中，我們將使用在 [入門步驟][First steps] 中構建的驗證器。但在開始之前，我們需要一些資金以及一對公/私鑰來持有這些資金。我們可以使用 PyCardano 生成私鑰和地址。

讓我們將第一個腳本撰寫為 `generate-credentials.py`：

```py filename="generate-credentials.py"
from pycardano import Address, Network, PaymentSigningKey, PaymentVerificationKey

signing_key = PaymentSigningKey.generate()
signing_key.save("me.sk")

verification_key = PaymentVerificationKey.from_signing_key(signing_key)

address = Address(payment_part=verification_key.hash(), network=Network.TESTNET)
with open("me.addr", "w") as f:
    f.write(str(address))
```

<Callout type="info">
  您可以透過以下方式執行上述指令：

```console
uv run generate-credentials.py
```
</Callout>

現在，您可以前往 [Cardano 水龍頭 (Cardano faucet)](https://docs.cardano.org/cardano-testnet/tools/faucet)，將一些 preview 網路的資金發送到新建立的地址（`me.addr` 內容）。

## 使用合約

現在我們有了資金，可以將它們鎖定在我們新建立的合約中。我們將使用 [PyCardano][] 透過 [Blockfrost][] 構建並提交交易。

<Callout type="info">
  這只是使用我們喜愛的工具進行設定的一個範例。如需更多工具，請務必查看 [Cardano 開發者門戶 (Cardano Developer Portal)](https://developers.cardano.org/tools)！
</Callout>

### 設定

首先，我們設定 PyCardano 並將 Blockfrost 作為提供者。這將允許我們讓 PyCardano 為我們處理交易構建，包括管理找零。它還為我們提供了一種稍後提交交易的直接方式。

在專案根目錄建立一個名為 `hello-world-lock.py` 的檔案，並添加以下程式碼：

```py filename="hello-world-lock.py" {7}
from pycardano import (
    BlockFrostChainContext,
)
import os

context = BlockFrostChainContext(
    project_id=os.environ["BLOCKFROST_PROJECT_ID"],
    base_url="https://cardano-preview.blockfrost.io/api/",
)
```

<Callout type="warning">
請注意，上述突顯行會尋找名為 `BLOCKFROST_PROJECT_ID` 的環境變數，其值必須設定為您的 Blockfrost 專案 ID。
您可以在終端機中透過執行以下命令定義新的環境變數（在執行腳本的同一個 session 中！）：

```console
export BLOCKFROST_PROJECT_ID=preview...
```

將 `preview...` 替換為您實際的專案 ID。
</Callout>

接下來，我們需要從之前生成的藍圖 (`plutus.json`) 中讀取驗證器。我們還需要將其轉換為 PyCardano 可理解的格式。這是透過序列化驗證器並將其轉換為十六進位文本字串來完成的：

```py filename="hello-world-lock.py" {3-5, 10-21, 28-30}
from pycardano import (
    BlockFrostChainContext,
    PaymentSigningKey,
    PlutusV3Script,
    ScriptHash,
)
import json
import os

def read_validator() -> dict:
    with open("plutus.json", "r") as f:
        validator = json.load(f)
    script_bytes = PlutusV3Script(
        bytes.fromhex(validator["validators"][0]["compiledCode"])
    )
    script_hash = ScriptHash(bytes.fromhex(validator["validators"][0]["hash"]))
    return {
        "type": "PlutusV3",
        "script_bytes": script_bytes,
        "script_hash": script_hash,
    }

# ... 之前的設定保持不變

signing_key = PaymentSigningKey.load("me.sk")

validator = read_validator()
```

### 將資金鎖定到合約中

現在我們可以讀取驗證器，開始進行第一筆交易將資金鎖定到合約中。Datum 必須符合驗證器所預期的表示形式（如藍圖中所指定的），因此它是一個帶有單個位元組陣列欄位的構成函數。

作為該位元組陣列的值，我們提供公鑰雜湊。這將是解鎖資金所必需的。

```py filename="hello-world-lock.py"
from dataclasses import dataclass
from pycardano import (
    Address,
    BlockFrostChainContext,
    Network,
    PaymentSigningKey,
    PaymentVerificationKey,
    PlutusData,
    PlutusV3Script,
    ScriptHash,
    TransactionBuilder,
    TransactionOutput,
)
from pycardano.hash import (
    VerificationKeyHash,
    TransactionId,
    ScriptHash,
)
import json
import os

# ... read_validator 函式定義 ...

def lock(
    amount: int,
    into: ScriptHash,
    datum: PlutusData,
    signing_key: PaymentSigningKey,
    context: BlockFrostChainContext,
) -> TransactionId:
    # 讀取地址
    with open("me.addr", "r") as f:
        input_address = Address.from_primitive(f.read())
    contract_address = Address(
        payment_part = into,
        network=Network.TESTNET,
    )

    # 構建交易
    builder = TransactionBuilder(context=context)
    builder.add_input_address(input_address)
    builder.add_output(
        TransactionOutput(
            address=contract_address,
            amount=amount,
            datum=datum,
        )
    )
    signed_tx = builder.build_and_sign(
        signing_keys=[signing_key],
        change_address=input_address,
    )

    # 提交交易
    context.submit_tx(signed_tx)
    return signed_tx.id

@dataclass
class HelloWorldDatum(PlutusData):
    CONSTR_ID = 0
    owner: bytes

# ... context 設定 ...

signing_key = PaymentSigningKey.load("me.sk")

validator = read_validator()

owner = PaymentVerificationKey.from_signing_key(signing_key).hash()

datum = HelloWorldDatum(owner=owner.to_primitive())

tx_hash = lock(
    amount=2_000_000,
    into=validator["script_hash"],
    datum=datum,
    signing_key=signing_key,
    context=context,
)

print(
    f"2 tADA 已鎖定到合約\n\t交易 ID: {tx_hash}\n\tDatum: {datum.to_cbor_hex()}"
)
```

<Callout type="info">
您可以透過執行以下命令執行上述腳本：

```

```
</Callout>

- 將 `hello-world-lock.py` 檔案放在 `hello-world` 資料夾根目錄。

### 從合約解鎖資金

最後一步：我們現在想要花費被 `hello-world` 合約鎖定的 UTxO。

要使交易有效，必須滿足兩個條件：

- 必須提供 "Hello, World!" 作為 redeemer；並且
- 必須由 datum 中引用的金鑰（即所有者）簽署。

讓我們建立一個新檔案 `hello-world-unlock.py` 並複製一些模板程式碼。

```py filename="hello-world-unlock.py"
from pycardano import (
    Address,
    BlockFrostChainContext,
    Network,
    PaymentSigningKey,
    PlutusV3Script,
    ScriptHash,
    UTxO,
)
import json
import os
import sys

# ... read_validator 函式定義 ...

def get_utxo_from_str(context, tx_id: str, contract_address: Address) -> UTxO:
    for utxo in context.utxos(str(contract_address)):
        if str(utxo.input.transaction_id) == tx_id:
            return utxo
    raise Exception(f"找不到交易 {tx_id} 的 UTxO")

# ... context 設定 ...

signing_key = PaymentSigningKey.load("me.sk")

validator = read_validator()

# 獲取要花費的 utxo
utxo = get_utxo_from_str(context, sys.argv[1], Address(
    payment_part = validator["script_hash"],
    network=Network.TESTNET,
))
```

最後，我們可以添加剩餘部分程式碼，並構建一個交易來解鎖我們之前鎖定的資金。這次我們需要構建一個 redeemer。

<Callout type="warning">
  請注意，我們需要使用 `.required_signers` 顯式添加簽署者，以便將其添加到交易的 `extra_signatories` 中 —— 並能被我們的腳本存取。

  此外，`.add_input_address` 對於確保交易構建器添加輸入以支付費用和抵押品是必要的。
</Callout>

```py filename="hello-world-unlock.py"
from dataclasses import dataclass
from pycardano import (
    Address,
    BlockFrostChainContext,
    Network,
    PaymentSigningKey,
    PaymentVerificationKey,
    PlutusData,
    PlutusV3Script,
    Redeemer,
    ScriptHash,
    TransactionBuilder,
    TransactionOutput,
    UTxO,
)
from pycardano.hash import (
    VerificationKeyHash,
    TransactionId,
    ScriptHash,
)
import json
import os
import sys

# ... read_validator 函式定義 ...

def unlock(
    utxo: UTxO,
    from_script: PlutusV3Script,
    redeemer: Redeemer,
    signing_key: PaymentSigningKey,
    owner: VerificationKeyHash,
    context: BlockFrostChainContext,
) -> TransactionId:
    # 讀取地址
    with open("me.addr", "r") as f:
        input_address = Address.from_primitive(f.read())

    # 構建交易
    builder = TransactionBuilder(context=context)
    builder.add_script_input(
        utxo=utxo,
        script=from_script,
        redeemer=redeemer,
    )
    builder.add_input_address(input_address)
    builder.add_output(
        TransactionOutput(
            address=input_address,
            amount=utxo.output.amount.coin,
        )
    )
    builder.required_signers = [owner]
    signed_tx = builder.build_and_sign(
        signing_keys=[signing_key],
        change_address=input_address,
    )

    # 提交交易
    context.submit_tx(signed_tx)
    return signed_tx.id

# ... get_utxo_from_str 等定義 ...

@dataclass
class HelloWorldRedeemer(PlutusData):
    CONSTR_ID = 0
    msg: bytes

# ... context, signing_key, validator setup ...

# 獲取要花費的 utxo (來自 argv[1])
utxo = get_utxo_from_str(context, sys.argv[1], Address(
    payment_part = validator["script_hash"],
    network=Network.TESTNET,
))

# 構建 redeemer
redeemer = Redeemer(data=HelloWorldRedeemer(msg=b"Hello, World!"))

# 執行交易
tx_hash = unlock(
    utxo=utxo,
    from_script=validator["script_bytes"],
    redeemer=redeemer,
    signing_key=signing_key,
    owner=PaymentVerificationKey.from_signing_key(signing_key).hash(),
    context=context,
)

print(
    f"2 tADA 已從合約解鎖\n\t交易 ID: {tx_hash}\n\tRedeemer: {redeemer.to_cbor_hex()}"
)
```

<Callout type="info">
像往常一樣執行此腳本，但這次還要傳遞鎖定資金之前命令獲得的交易 ID。例如：

